⚙️ Google API 연동 설정

구글 캘린더 일정을 자동으로 읽어오기 위한 API 자격증명을 입력하세요. 이 정보는 유출되지 않으며 오직 사용자 브라우저(LocalStorage)에만 안전하게 보관됩니다.

🛠️ 연동을 위해 구글 개발자 콘솔 설정 필요:
  1. Google Cloud Console에 접속해 새 프로젝트를 만듭니다.
  2. Google Calendar API를 사용하도록 활성화(Enable)합니다.
  3. OAuth 동의 화면을 구성하고, 테스팅 사용자로 본인 이메일을 등록합니다.
  4. 사용자 인증 정보 메뉴에서 OAuth 2.0 클라이언트 ID(웹 애플리케이션 유형)를 생성합니다.
  5. 승인된 JavaScript 원본(Origin)에 현재 접속 주소(예: http://localhost:3000, 로컬 실행 시 http://localhost)를 추가합니다.
